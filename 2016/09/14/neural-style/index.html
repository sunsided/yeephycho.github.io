Neural Style -- A Computer Artist

  • Convolutional Neural Network
  • Machine Learning
  • Neural Style
  • Tensorflow/Torch7

This post is an introduction to neural style program. Neural style is a CNN based algorithm to apply an image’s style to another image, this it the most interenting program which is related to deep learning I ever found. The source code is published on the Github, Torch7 version and tensorflow version.
The related paper is A Neural Algorithm of Artistic Style, published on Sep. 2015, and Artistic style transfer for videos, published on Apr. 2016.
